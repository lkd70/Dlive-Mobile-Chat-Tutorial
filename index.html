DLive Mobile Streaming: Monitoring Chat IRL

This short guide will walk you through step-by-step to get yuo started using an on-screen chat in Streamlabs for mobile.

First things first: TidyLabs

If you've already grabbed your TidyLabs URL and configured the Chat Overlay options, you can Skip Ahead to the next section!
Keep on scrollin'

  • Next

Getting your TidyLabs URL

We'll need a URL for the overlay widget, to obtain this, we'll navigate to TidyLabs and login with our Dlive account.

  • Next

Once signed in, head over to the Chat Overlay
page and select the "Copy URL" button.

Configuring Chat Overlay

Great, we've got our chat overlay URL! Now it's time to configure how it looks and acts.
First, we'll change the "Remove chat messages after X Seconds" option to "Never"
Next, let's be sure we've got a good strong dark background colour to make sure we can read our text:

  1. Change "Show Message Background" to "Yes"
  2. Set the "Background Colour" to something nice and dark, like black (#000000)

Don't forget to hit Save!

  • Next

Next up: Configuring StreamLabs

Now that we have TidyLabs all set up how we want it and we've copied our Chat Overlay URL, We can start looking at StreamLabs and configuring it to allow us to see the chat as we stream.

  • Next

Opening the Editor

In the StreamLabs app, swipe from the left-hand side of the screen to reviel the menu. From here, select the "Editor" option.

  • Next

Adding a widget

Now click the icon in the top-right corner to open the widget panel

From here, click the plus (+) icon to add a widget.
Select "Add Widget", then "Custom URL", and finally we can paste in the URL we were given by TidyLabs

  • Next

Making the widget visible

Great! Now that the widget is visible on our Editor, it's time to change the size, move it around, and most importantly: Make sure the stream can't see it but we can.
To do this, we have a few simple steps to follow:

  1. Click the widget we added until three icons appear next to it.
  2. Click the three dots icon and ensure the options at the bottom of the popup are set to "Hidden on Stream"> and "Showing on Preview">

We're all done!
